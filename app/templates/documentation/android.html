Initial Setup

When the app is first started, it will require the address of this OEE server.

Once connected to the server, the app will try and find its assigned machine. If it has not been paired to a machine, the IP of this tablet should be added to the machine in the admin page. The tablet will say its IP address at the top of the screen. If this address does not sound correct (e.g. 127.0.0.1) then it's possible you have an error in your server configuration. An example nginx config can be found in this project's repository

Normal Operation

At the login screen for the app, an operator can login with their user ID and password. The name of the assigned machine can be seen at the top of this view. User IDs can be found on the admin tab, where passwords can also be changed. The android app only accepts numerical passwords for ease of entry in suboptimal conditions.

After login, the user will be prompted to start a job on the assigned machine. The data entered here can be modified on request.

During a job the tablet displays the current state of the machine. This can be changed by tapping the dropdown

At the end of a job, the tablet will request additional data from the user. The data requested at this stage can be modified on request
